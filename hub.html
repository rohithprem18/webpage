Network Hub

A network hub is a basic networking device that connects multiple devices in a network, allowing them to communicate with each other. Unlike a switch, a hub sends data to all connected devices, regardless of the intended recipient, which can lead to network inefficiencies and collisions.

How Network Hubs Work

Network hubs work by receiving incoming data packets from one device and broadcasting them to all other connected devices. Each device on the network receives the data, regardless of whether it is the intended recipient. This broadcast method can lead to network congestion and reduced performance as the number of devices increases.

Advantages of Network Hubs

  • Simplicity: Hubs are simple devices that are easy to set up and configure.
  • Cost-Effective: They are generally less expensive than switches and other networking equipment.
  • Basic Connectivity: Ideal for small networks where advanced features are not required.
